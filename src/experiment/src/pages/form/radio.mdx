The Radio Component allows users to select one option from a set of mutually exclusive choices.

## Radio

<Tabs>
    <TabItem label="Preview">
        <Card>
            <div class="flex lg:w-1/2 mx-auto justify-center">
                <input type="radio" name="example" class="form-radio" checked/>
            </div>
        </Card>
    </TabItem>
    <TabItem label="HTML">
        ````xml
        <input type="radio" name="gender" class="form-radio" checked/>
        ````
    </TabItem>
    <TabItem label="JSX">
        ````xml
        <input type="radio" name="gender" className="form-radio" checked/>
        ````
    </TabItem>
</Tabs>

Each radio button in a group shares the same `name` attribute, ensuring that only one button can be selected at a time.
